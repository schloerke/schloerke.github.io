Theophylline, Warfarin & Monolix

Toy Data Sets

Theophylline, Warfarin, and the Monolix Data Sets are the standard data sets that will be used in this website for examples. Please click on them to see how the each data set was massaged to fit the needs of the functions/examples.

Info

  • Six data sets: Theophylline, Warfarin x2, and Monolix x3
  • warfarin.mdvpk.1 was used to generate the examples for the Variations
  • The Monolix data sets were used to generate the examples for Box Plot,Quantile Plot,Scatter Plot
  • Each data set was altered from its original condition

Requirements

  • All 'X' and 'Y' data must be ready to plot, with no filtering necessary, when sent to the plotting function
    • Please see the examples throughout the website
  • DOSE is repeated for each row of each ID
  • When the Administration happens, the AMT cell must equal the DOSE given (data$AMT==data$DOSE)
  • Split or subset the data before sending it to the function
    • Ex. Warfarin contains both MDVPK==0 and MDVPK==1
    • Warfarin was split into both warfarin.mdvpk.0 and warfarin.mdvpk.1
  • Check to make sure columns have not been repeated
    • NONMEM automatically attaches columns to the last four positions unless told not to, this could cause repeated columns
    • You might be wanting to use the second column named 'DV', but R interprets the second column as 'DV.1'
