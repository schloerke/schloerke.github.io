a-b-line

Variation

Use

  • To plot an abline
  • Good for Y~Y plots

Arguments

intercept
- Y intercept of Graph
slope
- Slope of line
black.and.white
- Logical, Describes whether the line should be for grey scale or color output

Notes

  • The same a-b-line will be plotted in each panel

Code

'plot.abline' <-
function(
  intercept = 0,
  slope = 1,
  black.and.white = black.and.white.in.panel
){
  cat("\nPlotting abline (",intercept,",",slope,")\n")
  
  if(black.and.white){
    line.color = "black"
    line.style = "dotted"
  }else{    
    line.color = "darkblue"
    line.style = "dotted"
  }
  line.thickness = 2

  panel.abline(
    a = intercept,
    b = slope,
    col = line.color,
    lwd = line.thickness,
    lty = line.style
  )
}
